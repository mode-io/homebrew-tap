# Mode IO Homebrew Tap

This repository is the public Homebrew tap for Mode IO command-line tools and packaged local apps.

Once published formulas are available, the intended install pattern is:

```bash
brew tap mode-io/tap
brew install skill-manager
```

`skill-manager` is planned as the first formula in this tap. Additional formulas can be added here as more Mode IO projects gain public Homebrew support.
